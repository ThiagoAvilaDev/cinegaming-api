# CineGaming API 🎮🎬

CineGaming API é um Projeto pessoal sem 
fim lucrativos que reflete os meus conhecimentos 
atuais focado no meu desenvolvimento como 
Dev o projeto é focado em consumo de coleção de 
mídias(filmes e jogos), utilizando rotas HTTP e boas práticas.
***
## *Tecnologias utilizadas*
* **Java 25**: Versão LTS Linguagem Principal do projeto.
* **Spring Boot 4.0.6**: Framework utilizado para criação da arquitetura.
* **Spring Data JPA**: Camada de persistência e comunicação com banco de dados com uso de paginação.
* **Spring Boot Validation**: Utilizado para validação de entrada de dados.
* **MySQL**: Banco de dados relacional.
* **Flyway**: Para migração e automatização e controle do banco de dados.
* **Maven**: Gerenciador de dependências e build do projeto.

***
## *Boas práticas*
* **SOLID**: Código focado em responsabilidade única. 
* **Padrão MVC(Model-View-Controller)**: Organização do projeto dividida em camadas.
* **Orientação a objetos**: Uso do encapsulamento para modelar a forma de como se deve interagir com as classes.
* **Padrão DTO (Data Transfer Object)**: Uso dos DTOs como preferência para não expor classes principais aos endpoints externos.
* **Testes Automatizados**: Implementação de testes unitários para Camadas de Service e Controller para garantindo a integridade da regra de negócio.
